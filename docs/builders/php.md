# PHP

## PHP 7

目录 `builders/php7`

构建`builder`镜像
```
make build
```

测试`builder`镜像
```
make test
```

发布`builder`镜像
```
make docker
```

使用`builder`镜像

```
s2i build [source dir] zdnscloud/s2i-php-builder [app image name]
```
